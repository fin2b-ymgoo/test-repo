## Prerequisites

1. Node.js
    - 노드 버전 관리를 위해 [nvm](https://github.com/nvm-sh/nvm)(Node Version Manager)을 사용

**Example:**
```sh
$ nvm install [version]
Now using node [version] (npm [npm-version])

$ nvm use
Found '[YOUR_PATH]/cfs.front/.nvmrc' with version [version]
Now using node [version] (npm [npm-version])
```

2. Install
    $ npm install

    

## Start & watch
    $ npm start

## Build for production
    $ npm run build
